# Week 1 – Research on AI-Assisted 4KB SRAM Design

## Objective

The objective of Week 1 was to understand the complete SRAM design workflow using OpenRAM and the SKY130 Process Design Kit (PDK). Rather than generating a working SRAM layout, the focus was on studying the architecture, compiler flow, memory organization, and open-source design tools.

---

## Topics Studied

- SRAM Architecture
- 6T SRAM Cell
- Read, Write and Hold Operations
- Precharge Circuit
- Sense Amplifier
- Write Driver
- Row Decoder
- Column Decoder
- Memory Array Organization
- OpenRAM Design Flow
- SKY130 PDK
- Technology Configuration
- Generated Design Files (GDS, LEF, LIB, SPICE)
- Pre-layout and Post-layout Flow
- Common Tool Errors and Debugging Strategies

---

## Deliverable

✔ IEEE Two-Column Research Report

## Learning Outcome

Week 1 provided a strong understanding of SRAM architecture, OpenRAM workflow, and the open-source tools used in memory design. This knowledge forms the foundation for the circuit implementation in the following weeks.
