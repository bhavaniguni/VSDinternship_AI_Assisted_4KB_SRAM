# 1. 6T SRAM Bitcell

## Introduction

The 6T SRAM (Six Transistor Static Random Access Memory) cell is the basic storage element used in SRAM memories.

A single SRAM cell stores **one bit** of data.

The cell is constructed using:

- 2 PMOS transistors
- 4 NMOS transistors

Total = **6 Transistors**

---

## Structure

The SRAM bitcell consists of:

- Two cross-coupled CMOS inverters
- Two NMOS access transistors
- Two Bitlines (BL and BLB)
- One Wordline (WL)

```
              BL             BLB
               |              |
           Access         Access
             NMOS           NMOS
               |              |
        -----------------------------
        |                           |
      Inverter                 Inverter
        |                           |
        -----------------------------
               |
              GND
```

---

## Working Principle

The two CMOS inverters are connected back-to-back.

They continuously reinforce each other's outputs.

Possible stored values are:

| Q | QB |
|---|----|
|1|0|
|0|1|

These values remain stored as long as power is available.

---

# 2. Hold Operation

During Hold mode:

- Wordline = LOW
- Access transistors remain OFF
- Bitlines are disconnected

The two cross-coupled inverters continuously maintain the stored data.

Example:

```
Q = 1
QB = 0
```

The stored value remains unchanged.

---

# 3. Read Operation

Before reading:

- BL = HIGH
- BLB = HIGH

These bitlines are precharged to VDD.

### Step 1

Wordline becomes HIGH.

The access transistors connect the SRAM cell to the bitlines.

### Step 2

If the cell stores:

```
Q = 1
QB = 0
```

then:

- BL remains HIGH
- BLB discharges slightly.

Example:

```
BL  = 1.8V

BLB = 1.75V
```

The voltage difference is very small.

### Step 3

The Sense Amplifier detects this small voltage difference and determines the stored bit.

---

# Read Disturb

During reading, the internal storage node becomes connected to the bitline.

This causes a slight disturbance in the stored voltage.

If the SRAM cell is not properly designed, the stored value may accidentally flip.

This unwanted effect is called **Read Disturb**.

Good SRAM designs maximize read stability.

---

# 4. Write Operation

To write data into SRAM:

The Write Driver forces complementary values onto the bitlines.

Example:

To write logic '1'

```
BL = 1

BLB = 0
```

Wordline becomes HIGH.

The access transistors connect the bitlines to the storage nodes.

The previous data is overwritten.

The new value is stored.

---

# Write Margin

Write Margin indicates how easily a new value can replace the previous value stored in the SRAM cell.

Higher Write Margin means:

- Easier writing
- Better write reliability

Lower Write Margin indicates poor write capability.

---

# 5. Butterfly Curve

The Butterfly Curve is used to measure SRAM stability.

It is generated by plotting the Voltage Transfer Characteristics (VTC) of the two cross-coupled inverters.

The largest square that fits inside the butterfly-shaped graph is called the **Static Noise Margin (SNM).**

Larger SNM indicates:

- Better stability
- Higher immunity to noise

Smaller SNM indicates lower stability.

---

# Static Noise Margin (SNM)

Static Noise Margin is defined as the maximum DC noise voltage that the SRAM cell can tolerate without changing its stored value.

Higher SNM

- Better cell stability
- Better data retention

Lower SNM

- Higher probability of data corruption
